CinemaPals

CinemaPals is an Android app that I made for my final project of my Mobile Development course. I love watching movies so I wanted to create a movie database that also made it easier to ask friends out to watch new movies. I decided to not publish the app onto the Google Playstore because the showtimes API that I used was on a trial basis. The app still works, but the features that showed nearby theaters and showtimes are not functioning.

Notes:
  • Movie database Android app that allowed users to easily find information on all movies
  • Allowed users to see nearby theaters, available movies and showtimes
  • Used Retrofit to efficiently call APIs with the benefit of being a Type-Safe Client
  • Came up with a user-friendly solution to ask friends out to a specific movie/showtime

Cardio Tracker

Cardio Tracker is a summer project that I worked on. It was my first Android app so I learned tons in the process! You can start a session and it starts tracing your movements. Since burned calories are different for every user, it takes your weight into account and calculates an estimate from the amount of steps you take.

Notes:
  • Implemented Google Maps to track and trace a user's current run
  • Real-time data is calculated for calories burned, distance travelled, and time elapsed
  • SQLite is used to store fitness sessions and statistics

EMQ Electronic Retail Store

EMQ is a fully functioning website. It was a group project for our CS160 Software Engineering course. It took roughly 2.5 months to complete. As a team, we followed Agile development methodologies. I faciliated team scrums and led our weekly class presentations to give the class updates of our progress.

EMQ Founded By: Alan Lam, Johnny Lui, Danny Mou, and David Ngo

Notes:
  • Used PHP, HTML5, Bootstrap and JavaScript
  • A MySQL database is used to store account email, passwords (hashed & salted), order information
  • Google Maps was used to simulate live delivery tracking

Spartan Agora is the first mobile app that my teammates and I ever made! It was developed using Ionic 2 which is a hybrid framework that allowed us to deploy on both iOS and Android. Our product manager gave us this project because it was essential for the bi-yearly SJSU Poster Expo. This event had presentations from people in the industry,

Created by: Aatif Shah, Amir Radman, David Ngo, Evan Kardos, and Katherine Soohoo

Notes:
  • Dynamic event mobile app created for SJSU Poster Expo 2016
  • 100+ attendees used the app during the expo (currently available on the Google Playstore)
  • Developed using AngularJS, Node.js, and Ionic 2
  • Connects to a live database using Cloudant & Bluemix
